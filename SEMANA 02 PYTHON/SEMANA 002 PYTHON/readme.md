# Semana 002 - Python

## Descripción

Se reforzaron conceptos básicos e intermedios de Python aplicados al Procesamiento de Lenguaje Natural (NLP).

Incluyó manejo de estructuras de datos, funciones y librerías útiles.

## Contenido

- Notebooks con ejercicios prácticos de Python.
- Repaso de listas, diccionarios y bucles.
- Uso de librerías fundamentales para NLP.

## Objetivo de Aprendizaje

Al finalizar esta semana, el estudiante podrá:

- Manipular estructuras de datos en Python.
- Escribir funciones para automatizar procesos.
- Utilizar librerías clave en proyectos de NLP.
