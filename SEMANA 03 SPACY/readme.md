# Semana 003 - SpaCy

## Descripción

Se introdujo el uso de la librería **SpaCy**, una de las más potentes para el NLP.

El objetivo fue aplicar procesamiento de texto, tokenización y análisis gramatical.

## Contenido

- Notebooks con ejemplos prácticos de SpaCy.
- Tokenización y lematización.
- Reconocimiento de entidades (NER).

## Objetivo de Aprendizaje

Al finalizar esta semana, el estudiante podrá:

- Procesar texto en SpaCy.
- Identificar entidades nombradas en un corpus.
- Comprender la importancia del preprocesamiento en NLP.
