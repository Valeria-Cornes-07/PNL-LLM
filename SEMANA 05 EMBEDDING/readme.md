# Semana 005 - Embedding

## Descripción

Se introdujo el concepto de **embeddings**, representaciones densas y continuas de palabras que capturan relaciones semánticas.

El objetivo fue comprender cómo los modelos modernos representan el lenguaje.

## Contenido

- Notebooks con ejemplos de embeddings.
- Comparación con representaciones tradicionales (BoW, TF-IDF).
- Visualización de relaciones entre palabras en espacios vectoriales.

## Objetivo de Aprendizaje

Al finalizar esta semana, el estudiante podrá:

- Explicar qué son los embeddings.
- Representar palabras en espacios vectoriales.
- Interpretar similitudes semánticas a partir de embeddings.
