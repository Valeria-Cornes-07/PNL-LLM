# Semana 001 - Web Scraping

## Descripción

Se trabajó con técnicas de extracción de información desde páginas web, aplicando herramientas de Python como `requests` y `BeautifulSoup`.

El objetivo fue aprender a obtener datos de forma automática para su posterior análisis.

## Contenido

- Notebooks con ejemplos de extracción de datos.
- Prácticas de parsing de HTML.
- Manejo de etiquetas y estructuras de páginas web.

## Objetivo de Aprendizaje

Al finalizar esta semana, el estudiante podrá:

- Realizar solicitudes HTTP con Python.
- Extraer información relevante desde páginas HTML.
- Preparar datos para análisis en NLP.
