# Semana 006 - Word2Vec

## Descripción

Se trabajó con el modelo **Word2Vec**, un enfoque de embeddings basado en redes neuronales.

El objetivo fue entrenar y analizar representaciones vectoriales de palabras.

## Contenido

- Notebooks con implementación de Word2Vec.
- Uso de CBOW y Skip-gram.
- Visualización de palabras relacionadas en el espacio vectorial.

## Objetivo de Aprendizaje

Al finalizar esta semana, el estudiante podrá:

- Entrenar embeddings con Word2Vec.
- Analizar relaciones entre palabras en el modelo.
- Aplicar Word2Vec en proyectos de NLP.
