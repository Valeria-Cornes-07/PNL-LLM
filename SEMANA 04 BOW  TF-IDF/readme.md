# Semana 004 - Bag of Words y TF-IDF

## Descripción

Se exploraron representaciones vectoriales básicas de texto: **Bag of Words (BoW)** y **TF-IDF**.

El objetivo fue transformar texto en vectores numéricos para análisis y modelado.

## Contenido

- Notebooks con implementación de BoW y TF-IDF.
- Comparación de resultados entre ambos enfoques.
- Visualización de frecuencias de palabras.

## Objetivo de Aprendizaje

Al finalizar esta semana, el estudiante podrá:

- Representar texto como vectores.
- Diferenciar entre BoW y TF-IDF.
- Preparar datos para modelos de Machine Learning.
